# node要精通

`Node.js`是一个能够在服务器端运行`JavaScript`的开放源代码、跨平台`JavaScript`运行环境。`Node.js`由`Node.js`基金会持有和维护，并与`Linux`基金会有合作关系。`Node.js`采用`Google`开发的`V8`运行代码，使用事件驱动、非阻塞和异步输入输出模型等技术来提高性能，可优化应用程序的传输量和规模。这些技术通常用于数据密集的即时应用程序。
